# mhelper

A little project for learning neural network programming.

The goal is to build an homemade multilayer perceptron (MLP) framework.

## How to run it

**Linux / MacOS**

```
$ python3 -m venv venv
$ source venv/bin/activate
$ pip install -r requirements.txt
$ python main.py
```

**Windows**

* shutdown
* install Linux
* see previous steps
